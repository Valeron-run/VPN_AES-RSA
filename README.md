# VPN с гибридным шифрованием (AES + RSA)

**VPN_AES-RSA** — это учебно-демонстрационный проект, реализующий простое VPN-соединение с использованием гибридной криптографии. Сервер и клиент на Java обмениваются данными, защищёнными с помощью симметричного шифрования (AES) и асимметричного для безопасной передачи ключей (RSA).

## Возможности

*   **Гибридное шифрование**: Безопасная передача сессионного ключа AES через RSA.
*   **Аутентификация клиента**: Простая проверка подлинности при подключении.
*   **Защищённый обмен сообщениями**: Все передаваемые данные шифруются алгоритмом AES.
*   **Многопользовательская поддержка**: Сервер может обслуживать несколько клиентских сессий одновременно (благодаря классу `ClientSession`).
*   **Модульная структура**: Криптографические утилиты, обработка авторизации и отправка сообщений вынесены в отдельные классы.

## Стек технологий

*   **Язык**: Java (версия 17 или выше)
*   **Криптография**: Встроенные библиотеки Java (JCA — Java Cryptography Architecture)
*   **Сеть**: Сокеты (java.net.Socket, java.net.ServerSocket)

## Установка и запуск

Для запуска проекта вам потребуется **Java Development Kit (JDK)** версии 17 или выше.
